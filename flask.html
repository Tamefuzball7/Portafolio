Proyecto flask basico

Formulario

Este formulario esta desarrollado con el frameWork flask de python , haciendo uso de las librerias virtualenv, mysql.connection, request, url_for , render_template, flash y jinja2. El Back-End funciona por medio de peticiones, El usuario al realizar una peticion por medio de la vista(Web) esta es enviada al Back-End y el backend responde su peticion devolviendo a la vista una raspuesta. Para la base de datos use MariaDB(SQL) con El SGBD (Sistema gestor de base de datos) Xampp

Creacion Base de la Datos

Para la creacion de mi Base de Datos use una tabla ,y 4 campos en los cuales esta mi campo de id que es Auto incrementable , Nombre, Email, Motivo y Mensaje que son de tipo varchar.
